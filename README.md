# 🤖 AI Universal Assistant Bot

An AI-powered chatbot built using Python, LangChain, LangGraph, and Google Gemini API.  
This bot can answer questions, summarize PDFs, generate Python code, and suggest creative ideas.

---

## 🚀 Features

- 💬 Smart AI Chatbot (Q&A)
- 📄 PDF Summarizer (extracts and reads documents)
- 🧠 Python Code Generator
- 💡 AI Project Idea Generator
- ⚡ Fast responses using Gemini 2.0 Flash

---

## 🛠️ Tech Stack

- Python 🐍
- Google Gemini API 🤖
- LangChain
- LangGraph
- PyPDF
- Google Colab
